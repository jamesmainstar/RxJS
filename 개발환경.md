#### Standard JS
- 실제 웹 표준 그룹에서 개발한 것이 아니지만 npm, github, mongoDB 와 같은 수천개의 프로젝트에서 사용됨
- 설정 파일을 작성하지 않고 추가 할 수 있다.
- 자동으로 설정에 맞게 변경해주는 기능이 있다.
- 리뷰 시간에 코딩 스타일을 잡는 것을 방지한다.
