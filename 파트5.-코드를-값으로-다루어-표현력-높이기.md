### 코드를 값으로 다루어 표현력 높이기
#### go
즉시 전달받은 값을 평가합니다.
```js
const go = (...args) => reduce((a, f) => f(a), args);
go(
  0,
  a => a + 1,
  a => a + 10,
  a => a + 100,
  console.log
);
// output: 111
```
#### pipe
```js
const pipe = (f, ...fs) => (...args) => go(f(...args), ...fs)

pipe(
  a => a + 1,
  a => a + 10,
  a => a + 100,
  console.log
)(0);
```

#### go를 사용하여 읽기 좋은 코드로 만들기
```js
go(
  products,
  products => filter(p => p.price < 20000, products),
  products => map(p => p.price, products),
  products => reduce(add, products)
);
```
