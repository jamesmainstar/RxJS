#### 범주론
- 수학에서 함수
  - 입력과 출력이 있어야 함
  - 합성이 가능해야 함
- 범주
  - 객체의 집합
  - 화살표
  - 정의역과 치역을 가르키는 화살표
  - 각 객체가 가지고 있는 항등 화살표
  - 한 화살표의 치역과 다른 화살표의 정의역이 같으면 합성할 수 있음 (f . g)(x) = f(g(x))
- 컴퓨터 프로그램 재앙 4가지를 범주론 관점으로 해결
  - 실패
    - 성공시 (f . g)(x) = f(g(x))
    - 실패시 (f . g)(x) = g(x), f(g(x)) => g(x)라는 의미는 f(x) = x와 같은 항등을 의미한다.
  - 의존성 : A => Pref(B)
  - 불확실성 : A => P(B), P는 멱집합이다.
  - 파괴성 : A => IO(B)
