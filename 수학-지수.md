지수(Exponent)는 반복된 곱셈 연산을 말한다.

#### 지수법칙(m, n이 자연수)
- a^m * a^n = a^(m + n)
- (a^m)^n = a^(m * n)
- (ab)^n = a^n * b^n
- (a / b)^n = a^n / b^n
- a^m / a^n
  - [m > n] a^(m - n)
  - [m = n] 1
  - [m < n] 1 / a^(n - m)

#### 지수의 정수로의 확장
자연수, 0, 음의 정수를 정수라고 하는 데, 0과 음의 정수의 지수법칙을 알아본다.

조건 : (a ≠ 0)
- a^0 = 1
- a^-n = 1 / a^n
