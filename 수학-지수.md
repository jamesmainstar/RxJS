지수(Exponent)는 반복된 곱셈 연산을 말한다.

#### 지수법칙(m, n이 자연수)
- a^m ⋅ a^n = a^(m + n)
- (a^m)^n = a^(m ⋅ n)
- (ab)^n = a^n ⋅ b^n
- (a / b)^n = a^n / b^n
- a^m / a^n
  - [m > n] a^(m - n)
  - [m = n] 1
  - [m < n] 1 / a^(n - m)

#### 지수의 정수로의 확장
자연수, 0, 음의 정수를 정수라고 하는 데, 0과 음의 정수의 지수법칙을 알아본다.

조건 : (a ≠ 0)
- a^0 = 1
  - a^m ⋅ a^0 = a^(m + 0) = a^m
- a^-n = 1 / a^n
  - a^n ⋅ a^-n = a^n-n
    - a^0 = 1
  - a^n ⋅ a^-n = 1
    - a^-n = 1 / a^n

#### 홀함수와 짝함수
홀함수는 `f(x) = x^홀수`를 말하며, f(-x) = -f(x) 성질을 같는다.
```
증명
f(x) = x^홀수
 = f(-x) = (-x)^홀수
 = (-1)^홀수 ⋅ (x)^홀수
 = -1 ⋅ x^홀수
 = -x^홀수
 = -f(x)
```
그래프로 보면 `f(a) = b`는 `(a, b)`, `f(-a) = -b`는 `(-a, -b)` 임으로 `y=f(x)`의 그래프는 원점대칭이다.

짝함수는 `f(x) = x^짝수`를 말하며, f(-x) = f(x) 성질을 같는다.
```
증명
f(x) = x^짝수
 = f(-x) = (-x)^짝수
 = (-1)^짝수 ⋅ (x)^짝수
 = 1 ⋅ x^짝수
 = x^짝수
 = f(x)
```
그래프로 보면 `f(a) = b`는 `(a, b)`, `f(-a) = b`는 `(-a, b)` 임으로 `y=f(x)`의 그래프는 Y축대칭이다.
