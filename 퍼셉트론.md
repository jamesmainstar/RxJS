#### 퍼셉트론 알고리즘
- 프랑크 로젠 블라트 | 1957
- 신경망의 기원
- 다수의 신호를 입력으로 받아, 하나의 신호를 출력한다.
- 출력은 1, 0이 있다.
- 1은 신호가 흐른다라는 의미
- 0은 신호가 흐르지 않는다다는 의미

뉴런에서 보내온 신호의 총합이 정해진 한계를 넘어설때 1을 출력한다. 한계값을 임계값이라 하며 쎄타로 나타낸다.

```js
const and = (x1, x2) => {
  const [w1, w2, theta] = [0.5, 0.5, 0.7]
  const result = w1 * x1 + w2 * x2;
  return result > theta ? 1 : 0;
}
console.log(
  and(0, 0),
  and(1, 0),
  and(0, 1),
  and(1, 1)
);
```
